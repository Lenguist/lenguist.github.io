# lenguist.github.io

Personal website for Maksym Bondarenko hosted on GitHub Pages.

## Overview

Static portfolio site featuring:
- About section with profile and bio
- Interactive timeline visualization of professional experiences and projects
- Explorations/projects showcase
- Random facts and writing links
- Terminal/hacker aesthetic with neon green color scheme

## Design

- **Style**: Black background with neon green (`#00f900`) accents, monospace fonts, ASCII-art navigation
- **Fonts**: Goldman (headers), Audiowide (subheads), Courier New (body)
- **Layout**: Single-page with anchor-linked sections

## Timeline

The Life updates section uses a CSS Grid-based timeline that plots experiences and projects chronologically from June 2018 to present. Items are positioned using a column-packing algorithm to avoid overlaps.

## Deployment

Push to `main` branch and GitHub Pages automatically deploys the site to `https://lenguist.github.io/`
